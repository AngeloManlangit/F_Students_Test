## Cloning & Running

### Cloning the Git Repository

```sh

```

### Downloading the Dependencies

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
